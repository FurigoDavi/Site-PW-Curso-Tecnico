Curso Técnico

Desenvolvimento de sistemas

O curso de Desenvolvimento de sistemas ensina sobre desenvolver sistemas computacionais utilizando ambientes de desenvolvimento e linguagens de programação específica, Além de algumas outras áreas, como Design e Banco de dados.

O curso de D.S (Desenvolvimento de Sistemas) é dividido em 6 máterias:

Análise e Projeto de Sistemas Design Digital Programação Web Fundamentos da Informatica Técnicas de Programação e Algoritmos
A.P.S D.D P.W F.I T.P.A
